# Portfolio Website

A modern, visually stunning portfolio website built with React, featuring an interactive fluid simulation background and smooth animations.

🔗 **[Davyd Voloshyn](https://www.davyd-voloshyn.com/)** 

## 🚀 Tech Stack

### Frontend
- **React 18** - Modern React with hooks
- **Vite** - Lightning-fast build tool and dev server
- **TailwindCSS 3** - Utility-first CSS framework
- **Framer Motion** - Production-ready animation library
- **Three.js** - WebGL fluid simulation rendering

### Styling
- **CSS Modules** - Component-scoped styles
- **Custom Animations** - Keyframe animations for glows and gradients
- **Responsive Design** - Mobile, tablet, and desktop breakpoints

### Tools & Libraries
- **Lucide React** - Beautiful, consistent icons
- **React Icons** - Additional icon sets (Simple Icons for tech logos)
- **react-helmet-async** - SEO and meta tag management
